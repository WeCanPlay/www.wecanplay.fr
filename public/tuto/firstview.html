Create our first view

Now, we create a view that will represent a round of a game. For more explanation on the view see http://wiki.wecanplay.fr/doku.php?id=fr:wcp.view

The constructor of the view need an array, with the entries init, loop and destroy. Each of these must contains function. Init is call at the begining of the view, this serve to set the default value of the variables and set the callback events. Loop will be called for each refresh of the canvas, in this one we will code the game engine, like motion and collision. Draw is obviously the function call for the display on the screen. Finally destroy define what will happen at the end of our game. 

var game = {
    init: function () {
    },
    loop: function () {
    },
    draw: function () {
    },
    destroy: function () {
    }
};
var gameView = new WCP.View(game);

To start the view, just use the function start of the object. 

gameView.start();

Now, the page must look like this: 

<!DOCTYPE html>
<html>

    <body>

        <script type="text/javascript">
            CANVAS_WIDTH = 200;
            CANVAS_HEIGHT = 200;
            WCP.setCanvas('canvas', CANVAS_WIDTH, CANVAS_HEIGHT);
            var game = {
                init: function () {
                },
                loop: function () {
                },
                draw: function () {
                },
                destroy: function () {
                }
            };
            var gameView = new WCP.View(game);
            gameView.start();
        </script>
    </body>
</html>
