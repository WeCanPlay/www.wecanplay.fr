Et maintenant, VIE !

Each state of sonic is describ by many images. If you call the state 'walk', WCP will print one by one each images of this state. 
var sonic = new WCP.Sprite(WCP.Assets.get('sonic'), 0, 5);
// Describ all states of our Sonic
sonic.animations({
    timeRate: 66,
    walk: [{
        x: 39,
        y: 0,
        width: 39,
        height: 40
    }, {
        x: 80,
        y: 80,
        width: 39,
        height: 40
    }],
    wait: [{
        x: 0,
        y: 40,
        width: 39,
        height: 40
    }, {...}]
};
// Call the 'wait' state
sonic.setAnimation("wait");
